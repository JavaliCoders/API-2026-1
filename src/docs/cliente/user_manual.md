# Manual do Usuário

## 1. Visão Geral do Sistema
- Plataforma web desenvolvida para centralizar o processo de compras da organização.
- Permite gerenciar pedidos, cotações, fornecedores e controle de estoque em um único sistema.
- Facilita a comunicação entre os setores envolvidos no processo de aquisição.

## 2. Abertura de Pedidos de Compra
- Funcionários podem registrar novos pedidos de compra no sistema.
- Cada pedido contém informações como produto, quantidade, justificativa e prioridade.
- Após a criação, o pedido entra no fluxo de análise e aprovação.

## 3. Gestão de Cotações
- O setor financeiro pode cadastrar e gerenciar cotações de diferentes fornecedores.
- É possível comparar valores, prazos de entrega e condições oferecidas.
- Documentos relacionados à cotação, como propostas ou notas fiscais, podem ser anexados ao sistema.

## 4. Acompanhamento do Processo
- Cada pedido possui um histórico detalhado de todas as ações realizadas.
- O sistema registra automaticamente eventos como:
  - aprovação de pedidos
  - rejeição de solicitações
  - alterações de status
  - confirmação de recebimento de produtos
- Isso garante rastreabilidade e transparência no processo de compras.

## 5. Dashboard Gerencial
- Gestores e diretores possuem acesso a um painel com indicadores do sistema.
- O dashboard apresenta informações importantes para análise e tomada de decisão, como:
  - desempenho de fornecedores
  - prazos de entrega
  - status dos pedidos
  - situação do estoque.

## 6. Benefícios do Sistema
- Centralização das informações de compras.
- Maior controle e rastreabilidade dos pedidos.
- Melhoria na análise de fornecedores.
- Apoio à tomada de decisão baseada em dados.
