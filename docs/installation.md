
# Manual de Instalação

## 1. Pré-requisitos
Antes de executar o projeto, certifique-se de ter instalado em seu computador o Git, o Java JDK versão 17 ou superior e o Maven. Também é recomendável utilizar uma IDE para desenvolvimento Java, como o Visual Studio Code ou o IntelliJ IDEA, para facilitar a execução e manutenção do projeto.

## 2. Clonar o repositório
Para obter o projeto em sua máquina, clone o repositório utilizando o seguinte comando:

git clone https://github.com/JavaliCoders/API-2026-1.git

## 3. Acessar a pasta do projeto
Após clonar o repositório, acesse a pasta do projeto com o comando:

cd API-2026-1

## 4. Instalar dependências
Com o projeto aberto na pasta correta, utilize o Maven para instalar todas as dependências necessárias executando o comando:

mvn clean install

## 5. Executar a aplicação
Depois que as dependências forem instaladas, a aplicação pode ser iniciada com o comando:

mvn spring-boot:run

## 6. Acessar a API
Após iniciar a aplicação, a API estará disponível localmente no seguinte endereço:

http://localhost:8080
